# Smart Campus Crisis Response System

## Overview

A ServiceNow-based emergency incident management platform designed for educational institutions. The system automates emergency classification, routing, and monitoring to improve campus safety.

## Features

* Emergency Incident Management
* Medical Emergency Routing
* Fire Emergency Routing
* Technical Issue Handling
* Harassment Report Routing
* Flow Designer Automation
* Priority-Based Incident Processing
* Dashboard Monitoring

## Technologies Used

* ServiceNow Zurich
* Flow Designer
* Business Rules
* Tables & Forms
* Dashboard Analytics

## Workflow

Student Reports Emergency

↓

Emergency Incident Created

↓

↓

↓

↓

↓

Dashboard Monitoring

## Project Outcome

The system automatically routes emergency incidents to the appropriate response team and updates incident status in real time.
